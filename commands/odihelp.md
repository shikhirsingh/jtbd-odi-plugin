# /odihelp — Where do I start?

Invoke the `odihelp` skill. Don't try to guess what the user already knows — ask:

1. **Do you have a clear job statement yet?** (no → /definejob; yes → continue)
2. **Do you have real customer interviews or survey data?** (no → rehearsal/synthetic; yes → /extractoutcomes or /computescores depending on which)
3. **What's your budget + timeline?** (4–8 wks + $20k+ → Full ODI; 2–3 wks + ~$5k → Lite ODI; <1 wk → rehearsal only)

Return the single command to run next + one paragraph explaining what it does.
